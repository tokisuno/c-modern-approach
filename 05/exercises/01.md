## 1c
```c
#include <stdio.h>

int main(void)
{
    int i, j, k;
    i = 3;
    j = 2;
    k = 1;

    printf("%d\n", i < j == j < k);

    return 0;
}
```
Comparison of if i < j yields the same boolean result as if j < k (true: 1);
## 1d
```c
#include <stdio.h>

int main(void)
{
    int i, j, k;
    i = 3;
    j = 4;
    k = 5;

    printf("%d\n", i % j + i < k);

    return 0;
}
```
If the result of i % j (3 mod 4 = 3) + i (3) is less than k (5), which returns 0 (false: 0);
