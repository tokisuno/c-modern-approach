## 2d
```c
#include <stdio.h>

int main(void)
{
    int i, j, k;
    i = 1; j = 2; k = 3;

    printf("%d\n", i < j || k);

    return 0;
}
```
Evaluates if i < j, if it fails it falls back to k. (true: 1);
